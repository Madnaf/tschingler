# Tschingler 🎵

A vibe-coded, progressive web app (PWA) soundboard for your daily needs. Play sounds, add your own via URL, and keep the vibes going.

**Deployed at:** [https://Madnaf.github.io/tschingler/](https://Madnaf.github.io/tschingler/)

## Features

- **Instant Playback:** Click a card to play the sound.
- **Customizable:** Add new sounds by providing a name and a direct URL to an audio file.
- **Persistent:** Your custom sounds are saved in your browser's local storage.
- **Offline Support:** Tries to cache sounds for offline use (PWA capabilities).
- **Reset:** A "Nuclear Option" button to wipe all data and restore defaults.
- **Vibe Coded:** Built entirely on vibes. No rigorous testing, just pure intuition and determination.

## How to Use

1.  **Play:** Click any tile.
2.  **Add:** Click the `+` button in the header. Enter a name and a direct link to an MP3/WAV file.
3.  **Delete:** Click the small `x` on a tile to remove it.
4.  **Reset:** Click the red trash can icon 🗑️ to clear all data and start fresh.
5.  **Refresh:** If things get weird, just refresh the page.

## Troubleshooting

- **Sound not playing?**
    - The URL might be broken or the hosting site (like MyInstants) might block hotlinking.
    - Try finding a direct link to a raw audio file (ending in `.mp3`, `.wav`, etc.).
    - If a sound stops working, delete it and add a new valid URL.
